Zelda: Guía de Juego

Requisitos

Para jugar, es necesario tener Python instalado en el sistema. Generalmente, el sistema operativo incluye componentes necesarios, así que solo se debe instalar si el juego no funciona al ejecutarlo. Además, se requieren pequeños conocimientos sobre el uso de la terminal, tanto en Linux como en Windows.

Descarga del Juego

Lo primero es descargar el juego desde GitHub. Puedes encontrarlo aquí.

Instrucciones de Ejecución

  1. Abrir la terminal y navegar hasta la ruta del juego con el comando: cd "ruta del archivo"
  2. Ejecutar el archivo principal con el comando: py "menuPrincipal.py"
  3. En el menú del juego, escribir la opción deseada. Se puede escribir "help" para ver la ayuda.
  4. Escribir "new game" para comenzar una partida, y se solicitará el nombre del jugador (por defecto: Link si no se escribe nada).
  5. Seguir las instrucciones de la historia escribiendo "continue".
  6. Aparecer en el mapa, donde se pueden realizar diversas acciones, como:

Acciones en el Juego

  • Move "dirección" "número de casillas": Para moverse.
  • kook: Para cocinar (cerca de una cocina).
  • show inventory "categoría": Filtrar las categorías del inventario, entre otras acciones.

A medida que se obtienen objetos en el juego, se abren varias posibilidades, como:

  • Equip "armas": Equiparse con distintas armas.
  • unequip "arma": Des-equipar armas.
  • Eat "comida": Comer alimentos para curarse, entre otras opciones.
